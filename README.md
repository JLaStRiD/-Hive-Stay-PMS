# 🏨 Home Crest Hotel PMS (MVP)

A modern Hotel Property Management System (PMS) built for Home Crest Hotel to streamline daily hotel operations including room management, booking, check-in/out, payments, and real-time dashboard monitoring.

This project serves as a Minimum Viable Product (MVP) for actual hotel deployment and future SaaS expansion.

---

## 🎯 Project Purpose

To replace manual hotel operations with a digital, centralized, and real-time system that improves efficiency, reduces booking errors, and enhances guest experience at Home Crest Hotel.

---

## 🚀 Core Features (MVP)

- 🔐 User Authentication (Admin / Staff / Cashier)
- 🛏 Room Management System
- 📅 Booking & Reservation System
- 🏨 Check-in / Check-out Flow
- 💳 Payment Tracking (Cash / POS-ready)
- 📊 Real-time Dashboard (Occupancy & Revenue)
- 👥 Role-Based Access Control
